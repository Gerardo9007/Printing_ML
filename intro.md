# 프로젝트 소개 — 인쇄 품질검사 AI

## 배경

라벨·포장 인쇄는 공정 특성상 두 종류의 물리적 원본을 사람이 눈으로 대조해 검사한다.

1. **인쇄판(인쇄물)** — 디자인 원본(AI/PDF)과 실제 인쇄 결과물의 문안이 일치하는지 확인
2. **목형(다이커팅 칼날)** — 설계 도면과 실제 제작된 목형 칼날의 형상이 일치하는지 확인

두 검사 모두 "정답(원본·도면)"이 이미 존재하고, 실제 결과물을 정답에 겹쳐서 어긋난 부분을 찾는 동일한 원리로 동작한다. 현재는 이 대조 작업을 검사자가 육안으로 수행하고 있어 속도·재현성·기록에 한계가 있다.

이 문서 세트는 이 두 검사를 컴퓨터 비전 AI로 자동화하는 프로젝트의 기획 문서다.

## 과제 정의

### 과제 ① 인쇄판 문안검사 AI

- **목적**: OCR·비전으로 인쇄판(또는 1차 인쇄물) 이미지와 디자인 원본을 비교하여 문안 정합성을 자동 검증
- **검출 대상**: 문자 누락, 오탈자, 점·기호(마침표 등) 누락, 잉크 번짐, 성분표시 누락·위치 오류
- **문제 유형**: 참조 기반 이상검출 = 이미지 정합(Registration) + OCR + 픽셀 차분(Diff)
- **치명 리스크**: 성분표시 누락·오탈자는 법적 리스크로 직결 → 미검출(누락) 0을 최우선 목표로 설정

### 과제 ② 목형 칼날 불량검사 AI

- **목적**: 비전으로 목형 이미지와 설계 기준 형상(CAD/도면)을 비교하여 칼날 상태·형상을 자동 검증
- **검출 대상**: 칼날 휨, 끊김(절단 불량), 위치 오차, 마모
- **문제 유형**: 이상탐지(Anomaly Detection) + 형상 정합 + 결함 세그멘테이션
- **치명 리스크**: 칼날 끊김·휨은 다이커팅(톰슨) 불량·재작업으로 직결 → 결함 미검출 0을 최우선 목표로 설정

## 공통 원리

두 과제 모두 아래 3단계 흐름을 공유한다.

```
정답(원본/도면)과 정밀 정렬(Registration)
        ↓
정답 대비 실제물의 차이 측정 (픽셀 차분 / 형상 diff, mm 단위)
        ↓
이상 위치·유형 자동 표시 및 판정
```

## 현재 상태

- 두 과제 모두 검사 이미지를 **일부 보유**하고 있으나, 실제 불량(NG) 샘플은 극히 희소하다.
- 이를 보완하기 위해 표준 촬영 지그 구축과 합성 결함 증강을 병행하는 데이터 전략이 필요하다 (상세는 `spec.md` 참고).
