데이터 방법론 개요

2개 과제의 데이터 관점 비교

구분① 인쇄판 문안 정합성 검사 AI② 목형 칼날 불량 검사 AI
과제 목적OCR·비전으로 인쇄판 이미지와 디자인 원본을 비교, 문안 정합성 자동 검증비전으로 목형 이미지와 기준 형상을 비교, 칼날 상태·형상 자동 검증
검출 대상문자 누락, 오탈자, 점(마침표·기호) 누락, 번짐, 성분표시 누락·위치오류칼날 휨, 끊김(절단 불량), 위치 오차, 마모
문제 유형참조기반 이상검출 = 이미지 정합(Registration) + OCR + 차분(Diff)이상탐지(Anomaly Detection) + 형상 정합 + 결함 세그멘테이션
핵심 데이터 난제불량 샘플 희소 / 원본-인쇄물 정합쌍 필요 / 미세 결함(점·자간)불량 실물 확보 극히 어려움 / 형상 다품종 / 마모는 연속량
권장 접근(데이터)합성 결함 대량 생성(원본에서 자동 라벨) + 실측 정합쌍 소량정상 다량 학습 기반 이상탐지 + 합성 결함 삽입 + 폐목형 재활용
치명 오류(우선순위)성분표시 누락·오탈자 → 법적 리스크. 재현율(누락 0) 최우선칼날 끊김·휨 → 톰슨 불량·재작업. 결함 재현율 최우선
보유 현황일부 검사 이미지 보유 → 표준 재촬영 + 합성 증강으로 확충일부 이미지 보유 → 지그 표준촬영 + 합성 결함으로 확충

예시로 보는 두 검사 — 정답과 실제를 겹쳐 다른 곳을 찾는다

두 과제 모두 원리는 같습니다. 정답(원본·도면)에 실제 이미지를 겹쳐 어긋난 곳을 자동으로 표시합니다.

흐린 초록 점선 = 설계 기준선(정답 위치), 진한 회색 = 실제 칼날, 빨간색 = AI가 잡아낸 이상.

핵심 — 불량 샘플은 실제로 거의 없습니다. 그래서 정답(원본·도면)에서 오탈자·끊김 같은 결함을 인위적으로 만들어(합성 증강) 학습량을 채우고, 목형은 정상을 잔뜩 학습시켜 '평소와 다른 모양'을 이상으로 잡습니다.

① 인쇄판 문안 정합성 검사 AI — 데이터 방법론

수집 → 증강 → 보정/정제 단계별 실행 방법 (OCR + 비전 정합/차분)

단계항목세부 실행 방법목적 / 근거우선순위·주의
수집정상(OK) 데이터승인 완료된 인쇄판·1차 출력물을 고해상도 스캔(텍스트 최소 600dpi). 제품·폰트·언어별로 폭넓게 확보정상 분포 학습·차분 기준 확보高
수집디자인 원본(GT)AI·PDF 벡터 원본을 동일 배율로 라스터화하여 '정답' 기준 이미지 생성참조기반 diff의 정답지高 / 버전 일치
수집정합쌍 구성동일 LOT의 (원본 ↔ 인쇄물) 페어를 1:1 매칭하여 저장정합·차분 학습의 핵심 단위高
수집실불량(NG) 샘플실제 발생 오탈자·누락·번짐 검사 반려품을 유형·좌표와 함께 아카이빙평가셋·현실성 검증용中 / test 집중
수집촬영 표준화조명·배경·해상도·정렬·ICC 색프로파일 고정. 스캐너/라인스캔카메라 지그 사용도메인 편차 제거高
수집메타데이터제품코드·LOT·인쇄판버전·원본링크·검사결과·불량유형·좌표 스키마화(MES 연계)Traceability·자동 라벨高
증강합성 오탈자·누락원본 텍스트에서 문자 삭제/치환/이동/자간변형을 프로그램 생성(라벨 자동 부여)불량 희소성 해결 — 최대 효과高
증강점·기호 누락마침표·쉼표·상표기호(®,™) 미세 요소를 삭제/축소해 합성미세 결함 검출력 강화高
증강번짐·잉크 확산가우시안 블러, 모폴로지 팽창, 잉크 blob·도트게인 시뮬레이션번짐 불량 재현中
증강인쇄·스캔 노이즈하프톤, 센서 노이즈, JPEG 아티팩트, 미세 얼룩 추가촬영 환경 강건성中
증강기하 지터(미세)±1~2° 회전, 소폭 원근/스케일로 정합 오차 모사Registration 강건성中 / 반전 금지
증강조명·색상 지터밝기·대비·화이트밸런스·채도 소폭 변화조명 편차 대응中
보정이미지 정합(Registration)특징점·호모그래피로 원본과 인쇄물 정밀 정렬 후 차분오검출(정렬오차) 제거의 핵심高
보정전처리 정규화데스큐(기울기 보정), 배경 제거, 해상도 통일, 명암·색상 정규화입력 일관성 확보高
보정문자 정규화유니코드 NFC 통일, 혼동쌍(O/0, l/1, ㅇ/0) 사전 구축OCR 오탈자 오판 감소中
정제라벨 노이즈 제거OCR 결과-사람 교차검증, 이중검수, 이상 라벨 필터링라벨 신뢰도 확보高
정제클래스 불균형 보정합성:실측 비율 조정, focal loss, 가중 샘플링희소 불량 학습 안정화中
정제데이터 분할제품·LOT 단위 train/val/test 분리(누수 방지). 실불량은 test 충분 확보성능 과대평가 방지高
